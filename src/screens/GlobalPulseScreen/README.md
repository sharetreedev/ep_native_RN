# GlobalPulseScreen

Aggregated view of how all users are feeling globally.

## Features
- **Pulse Grid**: A visual representation of global emotional states using the 2x2 grid (Energy vs. Pleasantness).
- **Active Pulses Overlay**: Shows the total number of check-ins contributing to the global view.
- **Trending Section**: Swipe up to see "Global Shifts" – emotions that are trending or growing in prevalence.
- **Paging Interface**: Vertical paging between the Grid and the Trending list.

## Navigation
- `EmotionDetail`: Long press an emotion on the grid or tap a trending item to see detailed insights about that emotion.
