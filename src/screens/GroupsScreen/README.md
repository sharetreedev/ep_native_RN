# GroupsScreen

Collective emotional overview for specific user groups (Family, Work, Friends, etc.).

## Features
- **Pulse Grid**: Visualizes the emotional pulse of the currently selected group.
- **Group Management**:
  - **Create**: Setup new groups.
  - **Group List**: Swipe up to see all groups the user is part of.
- **Detailed Group Trends**: Navigate to specific group pages to see historical data.
- **Mock Data**: Currently uses `MOCK_GROUP_DATA` for the pulse visualization.
