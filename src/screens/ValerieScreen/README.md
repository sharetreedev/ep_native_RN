# ValerieScreen

Valerie is the AI-powered wellbeing assistant, providing a safe space for users to check-in and express their feelings.

## Features
- **AI Chat Interface**: Real-time conversation with Valerie.
- **Support Call**: Quick access to a voice call with Valerie.
- **Glassmorphism Design**: Uses subtle shadows and surface colors to create a premium, calm chat experience.

## Key Components
- `ScrollView`: Holds the chat history with custom bubbles.
- `KeyboardAvoidingView`: Ensures the input field remains visible.
