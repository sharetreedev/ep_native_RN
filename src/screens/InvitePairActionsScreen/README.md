# InvitePairActionsScreen

Provides specific methods for inviting a new Pair.

## Features
- **Invitation Methods**:
  - **Invite via WhatsApp**: Direct deep-linking to WhatsApp.
  - **Invite via SMS/Link**: Generates a shareable URL.
- **Visual Callouts**: Important message highlighting the limit of 5 slots in the primary circle.

## Navigation
- `WhatsApp`: Opens external WhatsApp client.
- `Share`: Opens native share sheet.
- `Back`: Returns to intro or dashboard.
