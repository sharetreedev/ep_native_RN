# SupportRequestsScreen

Allows users to view and manage their formal requests for support from the wellbeing team.

## Features
- **Status Tracking**: Shows the ID, status (e.g., Pending), and type of each request.
- **New Request**: Call to action for initiating further support.
- **Clean List View**: Minimalist cards for scannable request history.

## Components
- `RootStackParamList`: Navigates back to the main flow.
