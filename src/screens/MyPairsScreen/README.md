# MyPairsScreen

Personal social circle view, showing the emotional states of those closest to the user.

## Features
- **Pulse Grid**: Shows where the user's Pairs fall on the emotional spectrum.
- **Your Circle List**: Swipe up to see a list of current Pairs and available invitation slots.
- **Invitation Slots**: 
  - Limit of 5 slots in the primary circle.
  - "Empty Slot" indicators with quick invite buttons.
- **Member Insights**: Tap on a Pair in the list to view their detailed profile and emotional history.

## Navigation
- `InvitePairIntro`: Start the flow to invite a new Pair.
- `UserProfile`: View detailed trends and check-in history for a specific Pair.
- `EmotionDetail`: Long press an emotion on the grid to see insights.
