# LessonScreen

Educational content viewer with integrated audio player functionality.

## Features
- **Lesson Content**: Displays lesson title, body text, and artwork.
- **Audio Player**: 
  - Progress bar with seek functionality (simulated).
  - Play/Pause controls.
  - Skip forward/backward.
  - Speed control.
  - Volume/Speaker toggle indicator.
- **Scrollable Layout**: Content is scrollable while the player remains fixed at the bottom.

## Navigation
- `Back`: Returns to the previous screen.
