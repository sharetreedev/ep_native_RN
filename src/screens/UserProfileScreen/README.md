# UserProfileScreen

A comprehensive view of a user's (or their Pair's) emotional wellbeing data.

## Features
- **Multi-Tab Dashboard**:
  - **Pulse**: Visualizes engagement/sentiment levels.
  - **Outlook**: Predicts future mood trends.
  - **Last 7 Days**: A vertical timeline of check-ins.
- **Pair Integration**: Special view for checking into a teammate's pulse if the user is viewing a Pair.
- **Profile Stats**: Summary cards for check-in rates and frequency.

## Visualizations
- `PulseChart`: Engagement over time.
- `OutlookChart`: Predictive trend analysis.
- `EmotionLegend`: Contextualizes the sentiment scale.
