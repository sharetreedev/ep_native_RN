# InvitePairIntroScreen

The starting point for the pairing flow, explaining the benefits of connecting with others.

## Features
- **Value Proposition**: Clear messaging about "The Power of Pairing".
- **Dynamic Interaction**: "Got it" button to proceed to the next step.
- **Glassmorphism UI**: Uses the modern Pulse 4.0 design language with blurred backgrounds and layered cards.

## Navigation
- `InvitePairActions`: Proceed to the actual invitation methods.
- `Back`: Return to the `MyPairs` dashboard.
